# ActiveTrack Website

This is a separate local-first website for ActiveTrack. It does not change the
Expo mobile app.

## Run It

From the ActiveTrack project folder, run:

```bash
npm run website
```

Then open:

```text
http://localhost:4173
```

The website uses Canvas in `main.js` for the animated background. Account and
history data are saved in the browser on this device, and History can be
exported as a JSON file.
